>This section will have two subparts : `Building the model` & `Implementing the API`

 1. Virtual environment and install dependencies
 2. Create Django project
 3. Update settings.py
 4. Migrations
 5. API
 6. Update apps.py
 7. Edit views.py
 8. Test API : Insomnia / ~~POSTMAN~~
